# Icon

A Vue wrapper for Baklava's `bl-icon` component for displaying icons.

## Basic Usage

```vue
<template>
  <Icon name="home" />
</template>

<script setup>
import { Icon } from "@baklavue/ui";
</script>
```

## Props

| Prop   | Type          | Default     | Description                  |
| ------ | ------------- | ----------- | ---------------------------- |
| `name` | `BaklavaIcon` | `undefined` | Icon name from Baklava icons |
