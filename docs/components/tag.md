# Tag

A Vue wrapper for Baklava's `bl-tag` component for displaying tags.

## Basic Usage

```vue
<template>
  <Tag>Vue</Tag>
</template>

<script setup>
import { Tag } from "@baklavue/ui";
</script>
```

## Props

| Prop      | Type         | Default     | Description |
| --------- | ------------ | ----------- | ----------- |
| `variant` | `TagVariant` | `undefined` | Tag variant |
