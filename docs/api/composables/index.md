# Composables API

API reference for BaklaVue composables. Composables are organized by category:

- [UI & Feedback](/api/composables/ui-feedback) — useNotification, useConfirmDialog, useDisclosure
- [Forms](/api/composables/forms) — useScrollToError, useZodForm, useFormState, useFieldArray, useFormPersistence, useStepperForm
- [Data & Utilities](/api/composables/data-utilities) — useFile, useClipboard, useLocalStorage, useSessionStorage, useCookie
- [Performance](/api/composables/performance) — useDebounceFn, useThrottleFn, useIntervalFn, useTimeoutFn, useRafFn
- [Browser APIs](/api/composables/browser-apis) — useBreakpoints, useElementSize, useContainerScroll, useSticky, useIntersectionObserver
- [Data Fetching](/api/composables/data-fetching) — useFetch
- [Theme & Layout](/api/composables/theme-layout) — useBaklavaTheme, useColorScheme, useThemePreset, usePagination
